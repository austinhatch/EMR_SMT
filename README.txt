Running EMR_SMT

EMR_SMT is a Java based program. Please ensure you have Java downloaded on your machine.

In order to run EMR_SMT, please have the following software packages installed in your Java Classpath:
1. https://commons.apache.org/proper/commons-math/download_math.cgi
2. https://commons.apache.org/proper/commons-lang/download_lang.cgi

Also, please ensure your machine has python installed:
https://www.python.org/downloads/

And that Z3.py is installed on your machine.
Instructions for Z3 can be found under the "Python" heading here: https://github.com/Z3Prover/z3

Once all of the necessary softwares have been installed, EMR_SMT can be ran via terminal command line from the EMR_SMT/src folder using:
javac *.java
java EMR_SMT

or by running the executable jar file EMR_SMT.jar on this page.
